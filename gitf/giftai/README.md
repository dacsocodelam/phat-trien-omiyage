# 🎁 GiftAI - あなた専用のギフトエキスパート

> **ギフト選びをプレッシャーから楽しさに変える** - AI で完璧なギフトを 60 秒で見つけよう

## 🎯 **コンセプト & ビジョン**

GiftAI は、ユーザーが大切な人に最適なギフトを素早く見つけることを支援する賢い AI プラットフォームです。選択のストレスを軽減し、ギフトにより多くの意味を与えます。

### **ターゲットオーディエンス**

- **メイン**: 25-40 歳の男性、忙しい、都市部在住
- **サブ**: オフィスワーカー、両親へのギフトを贈りたい子供たち、アイデアに困っている人

### **ペルソナ例: 田中浩二さん**

- 年齢: 30 歳、職業: プログラマー
- 課題: 彼女へのギフト選びがわからない、間違った選択を恐れる
- 利点: 60 秒で彼女が「確実に」喜ぶギフトを見つけられる

## 🚀 **セットアップ & インストール**

### **前提条件**

- Node.js 18+ & Yarn
- Ruby 3.3.6 & Rails 8.0.2
- Google Gemini API キー

### **バックエンドセットアップ**

```bash
cd backend
bundle install
rails db:create db:migrate db:seed
echo "GEMINI_API_KEY=your_api_key_here" > .env
rails s  # ポート3000で実行
```

### **フロントエンドセットアップ**

```bash
cd frontend
yarn install
yarn dev --port 3001  # ポート3001で実行
```
